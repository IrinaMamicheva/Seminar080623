# Краткое руководство по git
## 1. Основные команды
* git init - создает локальный репозиторий
* git commit - создает коммит
* git add - добавляет файл (-a в git commit - am - делает тоже самое)
* git branch - выводит на экран список веток, которые у нас есть
## 2. Ветки
* git log --graph - списки коммитов в виде графа/дерева
* git log --graph - выводит список коммитов в виде графа/дерева
